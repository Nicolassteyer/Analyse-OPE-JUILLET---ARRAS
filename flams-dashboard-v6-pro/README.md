# Flams Dashboard Pro

Application Node.js sans dépendances externes pour Render.

## Render
Build Command: `npm install`
Start Command: `npm start`
Root Directory: vide si `package.json` est à la racine.

## Disque persistant Render
Mount path recommandé:
`/opt/render/project/src/data`

## Fonctionnalités
- Import HTML 2026 et HTML 2025
- Sauvegarde serveur
- Analyse OPE JUILLET ARRAS
- KPI de base
- Comparatif clients 2026/2025
- Détail clients midi / soir
