
# Flams OPE Dashboard v3

Web app avec backend Node.js : le fichier HTML importé est sauvegardé côté serveur, donc les personnes qui ouvrent le lien voient les mêmes données.

## Lancer en local

```bash
npm install
npm start
```

Ouvrir http://localhost:3000

## Mettre en ligne

À déployer sur Render, Railway, Fly.io ou VPS Node.js.

Commande de démarrage :

```bash
npm start
```

Port : utiliser la variable `PORT` fournie par l'hébergeur.

Important : sur certains hébergeurs gratuits, le disque peut être temporaire. Pour conserver les HTML à long terme, choisir un disque persistant ou connecter une base/S3.
